📋 Información importante sobre tu plan ▼

🍽️ La mejor dieta es la que puedes mantener

La adherencia es el factor #1 que predice el éxito de una dieta (Dansinger et al., JAMA 2005). Las dietas demasiado restrictivas aumentan el hambre, la ansiedad y la probabilidad de atracones (Polivy & Herman, 1985). Este plan usa un déficit moderado y alimentos reales que puedes comer a largo plazo sin sufrimiento.

🚶 Caminar: tu mejor aliado

Caminar es la forma de ejercicio más segura y sostenible para cualquier edad. Una meta-análisis de 15 estudios demostró que cada 2.000 pasos/día adicionales reducen la mortalidad por todas las causas un 8-11% (Paluch et al., Lancet 2022). No necesitas correr ni levantar pesas para mejorar tu salud: solo caminar más de lo que lo haces ahora ya marca una diferencia significativa.

🔬 Base científica de este plan

Las recomendaciones se basan en:

  • Déficit 15-20%: Óptimo para perder grasa preservando músculo (Helms et al., 2014; Iraki et al., 2019)
  • Proteína 1.6-2.4 g/kg: Rango para preservar masa muscular en déficit (Jäger et al., ISSN 2017)
  • Entrenamiento de fuerza: Estímulo imprescindible para retener músculo (Morton et al., 2018)
  • Fórmula Mifflin-St Jeor: La más precisa para población general (Frankenfield et al., 2005)
  • NEAT y pasos: El gasto por actividad no-ejercicio es el componente más variable del TDEE (Levine et al., 2006)

⚠️ Aviso

Esta herramienta es orientativa. Las cantidades y macros son estimaciones. Consulta siempre con un médico y/o nutricionista antes de modificar tu alimentación, especialmente si tienes patologías, tomas medicación o estás embarazada.

Sobre esta web

🎯 Qué es

Una herramienta de planificación nutricional que ajusta automáticamente las cantidades de alimentos según tus calorías objetivo. Basada en un plan real de un entrenador personal certificado.

🏗️ Cómo está hecha

HTML + CSS + JavaScript puro — sin React, sin Angular, sin ningún framework. Una sola página que funciona completamente en tu navegador sin necesidad de servidor.

  • ~350 líneas de HTML — estructura semántica con onboarding, app y modales
  • ~2500 líneas de CSS — diseño responsive, tema oscuro/claro, variables CSS
  • ~800 líneas de JavaScript — datos, lógica de escalado, renderizado y persistencia

📐 Arquitectura

Single Page Application (SPA) estática. Todo se ejecuta en el navegador:

  • Los datos nutricionales están definidos como arrays de objetos en JS
  • Las cantidades se escalan con un ratio simple: kcal_elegidas / 2500
  • Cada cambio dispara un re-render selectivo de las secciones afectadas
  • El estado se persiste en localStorage (selecciones, datos TDEE, tema)

🧮 Calculadora TDEE

Usa la fórmula de Mifflin-St Jeor (la más precisa para población general según la ADA) para calcular tu gasto calórico. Después aplica un factor de actividad diaria y le suma las calorías estimadas del entreno.

🔬 Base científica

Las dosis de suplementación están basadas en:

  • ISSN Position Stands — la referencia en nutrición deportiva
  • Jäger et al. 2017 — creatina monohidrato
  • Meta-análisis 2024 — omega-3 (EPA+DHA) y recuperación
  • Dominguez et al. 2025 — magnesio y rendimiento
